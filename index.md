### 安装
最简单的安装方式是克隆并运行demo项目: [中文说明](demo), [演示项目github](https://github.com/digimakergo/dmdemo). 

### Digimaker提供哪些
- 服务端的Go库 
- 模板语言(用于输出html)
- 远程访问接口REST api
- 客户端的库(digimaker-ui, 基于react)用于开发web应用
- 后台管理客户端(eui, 基于react)用于后台管理
